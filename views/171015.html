  1. internet communications eqipment .
    1. 허브(Hub) - 한꺼번에 여러 대의 컴퓨터를 연결하는 장치로 각 회선을 통합적으로 관리.
    2. 리피터(Repeater) - 디지털 신호의 장거리 전송을위해 수신한 신호를 재생시키거나 출력 전압을 높여 전송하는 장치.
    3. 브릿지(bridge) - 물리적으로 다른 네트워크를 연결할 때 사용
    4. 라우터(Router) - 네트워크와 네트워크 간을 연결할 때 반드시 필요한 장비로 가장 최적의 IP경로를 설정하여 전송.
    5. 게이트웨이(GateWay) - 다른 네트워크에 데이터를 보내거나 다른 네트워크로부터 데이터를 받아들이는 출입구 역할.
    6. DNS 서버 : 문자로 된 도메인 이름을 컴퓨터가 인식할 수 있는 IP 주소.주소를 변환해 주는 서버.

  1. 소프트웨어 사용권.
    1. ad ware - 배너 광고를 보는 대가로 무료로 사용하는 소프트웨어.
    2. share ware - 기능 혹은 사용 기간에 제한을 두어 배포하는 소프트웨어.
    3. bundle - 특정한 하드웨어나 소프트웨어를 구매하였을 때 포함하여 주는 소프트웨어.
    4. free ware - 돈을 낸지 않고도 사용가능하고 다른 사람에게 전달해 줄 수 있는 소프트웨어.

  1. computer operating System(OS)
    1. 시스템의 모든 동작 상태를 관리하고 감독하는 제어프로그램의 핵심 프로그램을 슈퍼바이저(Supervisor)라 부른다.
    2. 운영체제는 컴퓨터가 동작하는 동안 주기억장치에 위치하게 된다.
    3. 키보드, 모니터, 디스크 드라이브 등 필수적인 주변장치들을 관리하는 BIOS를 포함한다.
    4. 운영체제는 사용자가 응용프로그램을 편리하게 사용하고, 하드웨어의 성능을 최적화 할 수 있도록 한다.
