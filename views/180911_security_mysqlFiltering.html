180911_security_mysqlFiltering

filtering?
: 문제가 될만한 정보를 사전에 미리 제거하는 것, 혹은 들어오면서 분별하는 것

escaping?
: 저장된 정보를 사용자에게 노출할 때 발생할 문제를 찾아가 제외,제거 시키는 것

백신의 정밀검사 : escaping
백신의 실시간검사 : filtering
mysqli api >>
인자로 들어온 sql injection 공격을 막는다. pasing
sql구문에 들어가는 변수들(input값, url parameter 등)

복수의 sql 명령을 실행할 때 사용하는 api 함수
mysql에서의 sql은 ddl,dml,dcl 후에 ;(세미콜론)으로
그 한 행을 마무리 짓는데, 여러 행을 세미콜론으로 구분지어
보내면 다중 명령이 실행된다. 이때 위의 api는 그 값을 return받는다.

사용자에게(혹은 주기적으로) 들어오는 데이터는 항상 의심의 눈초리로 지켜본다.
  • 개발자의 의도에 크나 큰 변수로 작용하는 부분이므로.
  • DB - > SW로 흘러가기 때문에 썩은 물이 퍼질 수 있다.
