180728_wordpress_설치(ing) ----- 180730

  1. wget http://wordpress.org/latest.tar.gz
    - var/www/ 경로 안에 압축파일을 설치한다.

  2. tar -zxvf latest.tar.gz
    - 폴더내에 압축해제한다.
    * wordpress폴더가 생성되는데, 이를 사용하기 위해선 웹접속 기본디렉토리[root directory]를 변경해야한다![default : /var/www/html] >> ?


  1. sudo nano /etc/apache2/sites-available/000-default.conf
    - 나노에디터를 사용하여 기본값을 변경해준다.

  2. DocumentRoot를 원하는 경로로 지정해준다.(ex. /var/www/html >> /var/www)

  3. sudo chown -R www-data /var/www
    - 권한을 설정해준다.

  4. sudo chmod -R 755 /var/www
    - 권한을 설정해준다.

  5. sudo /etc/apache2/apache2 restart
    - 변경한 설정이 적용될 수 있도록 재시작해준다.

  6. 사이트 이름, 관리자(사용자이름),패스워드등을 입력해준다.

  7. 웹페이지에서 부가작업 진행.
